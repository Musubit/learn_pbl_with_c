# 使用 C 语言进行项目式学习

本项目以 C 语言为基础，采用项目式学习方法，旨在通过实践掌握 C 语言的核心概念与编程技巧。每个项目均配有详细说明和代码示例，助你逐步深入理解 C 语言。项目构建由 Makefile 管理，确保代码可编译、可运行。文档部分使用 LaTeX 生成，提供清晰的项目说明与代码注释。开发过程中，使用 Git 进行版本控制，完整记录项目进展。
